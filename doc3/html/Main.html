Description

Com esta documentação pretendemos que se torne mais fácil a leitura e o entendimento do nosso código para fazer o jogo Bomberman. Aqui fica um pequeno resumo da lógica usada para fazer o jogo, e mais à frente abordamos todas as funções uma a uma. Esta parte do trabalho teve o objetivo de o tamanha do jogo para que este ocupe menos espaço quando guardado.
